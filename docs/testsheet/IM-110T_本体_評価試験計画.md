# IM-110T 本体 評価試験計画（コピペ用 v4）

> **使い方**: xlsx テンプレート (`IM-110T 設計検証・妥当性評価試験計画表.xlsx`) を別名保存して、各試験項目の 4 行ブロックに対し下記の 5 つのセル内容を順にコピペしてください。
>
> **コピペ先のセル位置 (1 項目 4 行ブロック)**:
> - 【C】試験項目 → C列 (3 行縦結合)
> - 【E】試験方法 → E列 (3 行縦結合)
> - 【H】合否基準 → H列 (3 行縦結合)
> - 【J】結果データ → J列 (3 行縦結合、評価時に記入)
> - 【C4】製品検査・試験規格 → C列 4 行目
> - 【E4】サンプル数・使用機器 → E列 4 行目

---

## ヘッダ部 (xlsx の上部に貼付)

### 入力・目的（B7 セル）
```
IM-110T 本体（電源・表示・操作・RS-232C 通信・EEPROM/RTC・筐体）に対し、製品設計仕様書 IM-110(仮) V05、製品規格一覧表 IM-110T、IM-100P 製品検査・試験規格／抜取り検査、IM-110_Probe 設計検証評価試験計画表 を入力として、試験方法、合否基準を決定し本体評価試験内容を決定する。
※プローブ単体性能（既存 xlsx）、プログラム機能評価（別途）、取説評価（別途）、BLE 通信（基板パターンのみで実装予定なし）は対象外。
```

### 試験対象機器
```
・試作本体：3 台
・試作プローブ：3 本（C 系試験用、プローブ評価と兼用可）
・ケーブル：標準 6m × 3 本 + 10m × 1 本
```

### 測定条件・評価条件
```
本評価表ではプローブ単体性能・プログラム評価・取説評価・BLE 通信を対象外とする。
暫定基準を採用した項目は本評価実施までに製品要求仕様書／製品設計仕様書／製品規格一覧表で確定値を反映する。
```
