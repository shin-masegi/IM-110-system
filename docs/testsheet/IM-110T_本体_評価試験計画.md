# IM-110T 本体 評価試験計画 (v6 — 出典原文ベース、本体システム評価)

> **方針**: 本体評価の試験項目はプローブ単体評価計画 (IM-110T 設計検証・妥当性評価試験計画表.xlsx) の 16 項目をすべて本体システム (本体 + ケーブル + プローブ) で再実施する。試験基準・試験内容・試験方法は出典の原文 (プローブ計画表、または設計仕様書 V05) のまま使用。
>
> **コピペ先のセル位置 (xlsx テンプレート流用)**:
> - 【C】試験項目 → C列
> - 【E】試験方法 → E列 (試験方法 ① ② ...)
> - 【H】合否基準 → H列
> - 【J】結果データ → J列 (評価時に記入)
> - 【C4】製品検査・試験規格 → C列 4 行目 (規格 No.)
> - 【E4】サンプル数・使用機器 → E列 4 行目

---

## 1. 入力・目的

### 入力
- IM-110T 設計検証・妥当性評価試験計画表 (プローブ単体評価)
- 製品設計仕様書 IM-110(仮) V05
- 製品要求仕様書
- 製品規格一覧表
- IM-100P 抜取り検査チェックシート (検査-3308-11)

### 目的
IM-110T 本体システム (本体基板 + ケーブル + 成型プローブ) に対し、製品設計仕様書 IM-110 (V05)・製品要求仕様書・各製品検査試験規格を入力として、試験方法・合否基準を決定し評価試験内容を決定する。

プローブ単体評価計画 (項目 1〜16) を本評価でも実施し、加えて設計仕様書 V05 で本体評価が明示された本体固有項目 (品名・電池・ケーブル通信・電源変動・界面判断 等) を追加評価する。

### 対象外
- プログラム機能評価 (別途プログラム評価表)
- 取説評価 (別途取説レビュー)
- BLE 通信 (設計仕様書 No.134: 「基板パターンを設けておく」が「現段階で Bluetooth 搭載の予定はない」)

---

## 2. 試験対象機器

```
IM-110T 試作器 (本体 + ケーブル + 成型プローブ): 3 セット
```

**出典**: IM-110T 設計検証・妥当性評価試験計画表 項目 5

```
温度特性

【入力】
製品設計仕様書№55

IM-100P

【目的】
IM成型プローブの温度特性によるバラつき具合と補正方法の検討と効果の確認
```

**【E】試験方法**
```
MLSS/SS/透視度
①測定する標準液が入ったビーカーを5,20,35℃の恒温水槽に浸漬し、スターラーで1h以上撹拌する。
※蒸発により濃度が変わらないよう、サランラップなどで蓋をしておく
②測定する水槽にプローブを20分以上浸漬し、プローブ温度を安定させる。
③各温度、各濃度の標準液を測定する。
※透視度については、成型プローブで測定後、ガラス透視度計で測定する。
⓸20℃の測定値と5，35℃の測定値の誤差を確認する。

水深
①プローブを20℃恒温槽にて1h以上安定させる。
②ガスボンベにて0m,6m相当の圧力を加え、表示値を確認する。
③恒温槽の温度を5℃、35℃に変化させ、①、②を繰り返す。
```
