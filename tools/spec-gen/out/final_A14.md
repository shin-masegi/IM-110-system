透視度校正には以下の手順が含まれます。

1. **低レンジゼロ調整**
   - タイトル: 「低レンジゼロ調整」
   - 表示内容: 「開始」、単位「mV」および、センサーの入力電圧
   - 調整方法: センサーダミー入力無しの state with the record switch pressed, it calculates and stores the zero adjustment coefficient for the low range in internal memory. Once completed, the "Start" display is turned off.
   - Adjustment not possible if input voltage deviates by more than ±10% from each range span value.

2. **低レンジ調整**
   - タイトル: 「低レンジ adjustment」
   - 表示内容: 「開始」、単位「mV」および, sensor input voltage
   - 調整方法: Connect the sensor dummy input and press the record switch to calculate and store the span adjustment coefficient for the low range in internal memory. Once completed, turn off the "Start" display.
   - Adjustment not possible if input voltage deviates by more than ±10% from each range span value.

3. **中レンジゼロ調整**
   - タイトル: 「中レンジゼro adjustment」
   - 表示 content: "Start", unit "mV" and sensor input voltage
   - Adjustment method: Connect the sensor dummy input, press the record switch to calculate and store the zero adjustment coefficient for the mid-range in internal memory. Once completed, turn off the "Start" display.
   - Adjustment not possible if input voltage deviates by more than ±10% from each range span value.

4. **中レンジ調整**
   - タイトル: 「中レンジ adjustment」
   - 表示 content: "Start", unit "mV" and sensor input voltage
   - Adjustment method: Connect the sensor dummy input, press the record switch to calculate and store the span adjustment coefficient for the mid-range in internal memory. Once completed, turn off the "Start" display.
   - Adjustment not possible if input voltage deviates by more than ±10% from each range span value.

5. **高レンジゼro adjustment**
   - タイトル: 「高レンジZero adjustment」
   - 表示 content: "Start", unit "mV" and sensor input voltage
   - Adjustment method: Connect the sensor dummy input, press the record switch to calculate and store the zero adjustment coefficient for the high range in internal memory. Once completed, turn off the "Start" display.
   - Adjustment not possible if input voltage deviates by more than ±10% from each range span value.

6. **高レンジ調整**
   - タイトル: 「高レンジ adjustment」
   - 表示 content: "Start", unit "mV" and sensor input voltage
   - Adjustment method: Connect the sensor dummy input, press the record switch to calculate and store the span adjustment coefficient for the high range in internal memory. Once completed, turn off the "Start" display.
   - Adjustment not possible if input voltage deviates by more than ±10% from each range span value.

通常はこの時点で電源をOFFし基板調整を終える。
