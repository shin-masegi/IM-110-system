## 10 エラー表示（本体故障・バッテリエラー）

### 本体故障表示
EEPROMやRTCの読み書きに失敗した場合、または電源ON直後の初期化中に異常が検出された場合、以下の画面を表示します。

```
本体故障
EEPROM/RTCエラー
```

この画面は2秒間表示されると自動で解除され、元の画面に戻ります。履歴スイッチを押すと、メッセージとQRコードの代わりに処置手順を表示します。処置方法表示状態で再度履歴スイッチを押すと、処置手順の代わりにメッセージとQRコードを表示します。

### バッテリエラー表示
電池電圧が3.0V以下になった場合、以下の画面を表示します。

```
バッテリーエラー
電池交換が必要です
```

この画面は2秒間表示されると自動で解除され、元の画面に戻ります。履歴スイッチを押すと、メッセージとQRコードの代わりに処置手順を表示します。処置方法表示状態で再度履歴スイッチを押すと、処置手順の代わりにメッセージとQRコードを表示します。

### 測定中止動作
本体故障やバッテリエラーが発生した場合、測定は自動的に中止されます。記録スイッチを押すと、元の画面に戻りますが、測定は再開されません。電源をOFFし、適切な対処を行った後、再度電源をONして使用してください。

### 注意事項
- 本体故障やバッテリエラーが発生した場合は、すぐに対処することが推奨されます。
- バッテリエラーの場合、新しい電池に交換してから再度使用してください。
