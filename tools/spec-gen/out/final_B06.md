プローブが取得した生 mV から物理量（MLSS / SS / 透視度）への換算、すなわち濃度換算はすべて本体側で実行する。プローブが担うのは、生 AD 値の span 正規化（出力 = 生 mV × ADC_SPAN_INI / ADC_Span, ADC_SPAN_INI = 1700）と気圧の取得までであり、物理量への補正式そのものはプローブのアプリケーションでは展開しない。

補正に用いる全係数と調整パラメータは、プローブ側 flash の統合ストア（probe_store_t）を唯一の真実源として保管する。本体側 EEPROM はこのストアとバイト完全同一のミラー（非権威キャッシュ）であり、本体基板のみの交換でもプローブの付け替えでも測定値が正しく出るよう、3層モデルで整合を保つ。係数・パラメータの本体⇔プローブ転送は 32B ページ単位で行い、読み出しは RPG、ページ単位の書き込みは WPG、封緘・flash 確定は WSC を用いる。per-field の float テキスト化はせず、共有 struct の生バイト列をそのまま転送するため往復誤差は生じない。

調整パラメータは物理量換算式とは別レイヤに位置づけられ、測定フロントエンドの動作点を決めるものである。代表例として、LED duty（SP コマンド）は LED_REFV の PWM duty を設定して AD 入力レンジを整え、移動平均回数（SADA）はノイズ抑制の平均段数を、有効 ch マスク（SADC）は更新対象 ch を決める。これらは係数（濃度換算式のパラメータ）とは区別して扱う。

具体的な補正式（出荷時2次式、TR 累乗、ゼロ・温度補正、k_No 等）およびページ意味論・校正手順の詳細は、付録 A と校正手順（mlss-calc-reference.md §8 / §12、統合ストア各ページ定義）を参照すること。本節では個々の計算式は展開しない。
