本体⇔プローブ間の RS-232C 通信で使用する現行コマンドの一覧を以下に示す。全コマンドは本体（マスター）からプローブ方向にのみ発行され、コマンド名は大文字（A〜Z）のみ有効である。小文字・大小文字混在はすべて NG として扱われる。各フレームの終端には CR+LF（`\r\n`）を付与する。

| # | コマンド | パラメータ | 応答 | 機能 |
|---|---|---|---|---|
| 1 | VR | なし | 値応答1行 | 製品名・FW バージョン取得（例: `IM-110 Probe Ver.0.11`） |
| 2 | SP | X.XX（float, 0.0〜1.0） | OK / NG | LED_REFV の PWM duty 設定 |
| 3 | SEL | N（0,1,2） | OK / NG | SEL（PA6）出力モード（0:OFF / 1:ON / 2:PWM） |
| 4 | SS | N（0,1） | OK / NG | SEL3（PA7）出力（0:OFF / 1:ON）。ADC ch3 の読取先を AD_AVE_mV[2]（SEL3 LOW / PGA x1）と AD_AVE_mV[4]（SEL3 HIGH / PGA x2）で切替 |
| 5 | MS / MS,N | N（0,1）または省略 | OK | 測定データ自律送信の制御。MS,1=開始 / MS,0=停止（冪等）。引数省略時はトグル（レガシー互換） |
| 6 | MD | なし | 値応答1行 | 現在の測定値を1回送信（mV 5個＋気圧 hPa の float 6個） |
| — | MDR | なし | 値応答1行（`M` 始まり） | 生 AD 値 mV（span 非依存・移動平均前）を1回取得（調整・デバッグ用, §3.2.7） |
| 7 | SID | N（uint32 10進, 0〜4294967295） | OK / NG | Probe ID 設定 |
| 8 | SADZ | N（0〜4） | OK / NG | 【2026-07-21 廃止扱い・残置】ADC_Zero を設定するが計算では未使用 |
| 9 | SADS | N（0〜4） | OK / NG | MLSS AD0（スパン基準）。指定 ch の空中生 mV を ADC_Span に記録し、空中出力を 1700mV にする。取得ゲート 生 mV 1400〜2048mV、範囲外は NG |
| 10 | SADA | N（0〜255） | OK / NG | ADC 移動平均回数設定（既定 30） |
| 11 | SADC | XXXX（4文字 '0'/'1'） | OK / NG | ADC 有効 ch ビットマスク（MCP3424 CH1〜CH4） |
| 12 | RPP | なし | 複数行応答 | Flash パラメータ一覧出力 |
| 13 | WPP | なし | 固定文字列 | 現在のパラメータを内蔵 Flash に保存 |
| 14 | RPI | なし | 固定文字列 | パラメータを初期値にリセット（RAM 上のみ、永続化には続けて WPP が必要） |
| — | RPF | なし | OK / NG | 【保守・製造用】統合ストア（probe_store_t 512B）を既定値で初期化し flash まで確定する新品化リセット（§3.2.7） |
| — | RPG | なし または N（0〜15） | 複数行応答（PGV/PG/OK）または PG 1行＋OK | 統合ストアの 32B ページを読み出す。全16ページ一括または単一ページ。ストア無効時は NG（§3.2.6） |
| — | WPG | N（0〜15）,64hex | OK / NG | 統合ストア RAM 像の page N を更新（page 末尾 XOR を検証, §3.2.6） |
| — | WSC | なし | OK / NG | 統合ストア RAM 像を封緘（magic/ver/checksum/全ページ XOR）し flash へ確定する commit（§3.2.6） |
| M | FUP,45063 | magic（固定値 45063） | OK→ブートローダー移行 / NG | 【リペア・製造用】ROM ブートローダーへ移行し UART 書換可能状態にする（§3.2.5） |

上記いずれのコマンドにもマッチしない受信行、およびパラメータの欠落・形式不正・範囲外の場合は、プローブは NG を返送する。
