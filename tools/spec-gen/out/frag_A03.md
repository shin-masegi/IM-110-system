# 概略・システム構成

本システムは、本体とプローブの2つの基板から構成されています。以下にその概要とシステム構成について説明します。

## システム構成図

```plaintext
+-------------------+
|                   |
|      本体         |
|                   |
+---------+---------+
          |
          | RS-232C
          |
+---------v---------+
|                   |
|      プローブ       |
|                   |
+-------------------+
```

## 概要

### 本体とプローブの関係
本システムでは、本体がマスターとして機能し、プローブはスレーブとして動作します。プローブは本体から給電を受け取り、単独で動作することはできません。

### 接続方法
本体とプローブはRS-232Cケーブルを介して接続されます。この接続により、本体がプローブに対してコマンドを送信し、データのやり取りを行います。

## ハードウェア構成

### 本体
本体は以下の機能を持っています：
- マスターとしてプローブを制御する。
- プローブへの給電を行う。
- RS-232Cインターフェースを通じてデータ通信を行う。

### プローブ
プローブは以下の特徴を持ちます：
- 本体から給電を受け取り、動作します。
- 単独で動作することはできません。
- RS-232Cインターフェースを通じて本体とデータ通信を行います。

## まとめ
本システムは、本体がマスターとしてプローブを制御し、RS-232Cケーブルを介してデータ通信を行う構成です。プローブは本体から給電を受け取り、単独で動作することはできません。このような構成により、安定したデータ通信と制御が可能となります。
