# 8. BLE通信（Readerとの通信）

RN4871経由のBLEでReader(アプリ)と通信する動作概要は以下の通りです。

## 接続時のBTアイコン表示
接続中には、デバイスの画面にBTアイコンが表示されます。これにより、ユーザーは現在BLE接続が有効であることを視覚的に確認できます。

## 消費電流増加
BLE通信中は消費電流が増加します。これは通信のための追加の電力が必要となるためです。これにより、バッテリーの寿命に影響を与える可能性があります。

## 通信可能なデータ種別の概要
以下は、BLE通信でReaderと交換できる主要なデータ種別です：

- **測定値**: MLSS / SS / 透視度 / 界面などの現在の測定値。
- **校正情報**: スパン校正値、ゼロ校正値など。
- **エラー履歴**: 過去に発生したエラーや警告の履歴。
- **設定情報**: デバイスの設定情報（例：測定モード、校正設定など）。
- **バージョン情報**: ファームウェアバージョン、ハードウェアバージョンなど。

具体的なコマンドは本体FWと突き合わせ校正で確定します。
