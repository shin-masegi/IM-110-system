## 5.6 界面測定画面

### 仕様
界面(汚泥界面レベル)測定画面 is displayed with the following items:

- **Measurement Item**: Interface Level
- **Unit**: mm
- **Range**: 0 - 1000 mm (Note: Actual probe specification supports up to 500 mm)

### Details
The interface measurement screen displays the level of the sludge interface in millimeters. The measurement range is from 0 to 1000 mm.

### Notes
- If the measured value is out of range, an error message will be displayed.
- Appropriate error handling will be performed if any abnormality occurs during measurement.
