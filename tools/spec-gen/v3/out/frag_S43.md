ロガー履歴表示画面は、フラッシュに記録済みのデータロガー測定の一覧を確認するための画面である。`disp_DISPLOG()` によって描画され、`operation_mode` が `DISPLOG_2`（描画）および `DISPLOG_3`（表示中）の状態に対応する。ID-200T 由来の機能である。

画面上部にはロガー履歴表示用のタイトル（`t_logger_disp`）・電池残量アイコン・日付時刻を表示する。画面下部には履歴スイッチ（DISP）側に「切替」の操作案内アイコンを表示する。記録スイッチ（MEM）側の「次へ」案内はコード上で無効化されており、表示されない。

一覧は 1 画面に 3 件分を上下に並べて表示する。各件について、通し番号（01・02・03 の固定表示）、記録の開始日時（年／月／日 時：分）、測定間隔（分）、およびサンプル数（点）を表示する。表示する 3 件は、あらかじめ新しい順に並べ替えたインデックス（`log_sort_index[0]`〜`[2]`）で指定された履歴ヘッダ（`log_head`）から読み出す。測定間隔が 0 の履歴についてはサンプル数を 0 点として表示する。

この画面の表示中（`DISPLOG_3`）に履歴スイッチ（DISP）を押すと、ロガー測定の画面へ戻る。戻り先は、直前に測定を完了しているか（`log_comp_flag`）によって決まり、完了している場合はロガー測定完了画面、それ以外の場合はロガー測定待機画面へ遷移する。本画面では記録スイッチ（MEM）に対する処理は設けられておらず、記録スイッチ（MEM）を操作しても画面遷移や動作の変化は生じない。
