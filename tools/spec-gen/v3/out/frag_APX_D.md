### 調整機能とパラメータ記憶

#### 調整機能
1. **LED PWM 調整**
   - 目的: 空中で ch1〜4 の AD が 2048mV に達しないよう LED 出力を下げる。
   - 基準: ch1〜4 の生 mV で最も高いものが 1750mV になるよう LED duty を調整。
   - 保存先: `LED_Out[0]`（第1領域 PARAM）。
   - 対象 ch = ch1〜4（code index `[0]`〜`[3]`）。

2. **ADZero 個別オフセット**
   - ADZero = 空気中で受光出力を 1750mV にオフセットする値。
   - 対象 ch = ch1〜5（index `[0]`〜`[4]`）を個別に。
   - 取得ゲート: 生 mV 1400〜2048mV。

3. **Ref 温度補正**
   - 温度補正は 5/20/35℃ 清水 Ref → 2次式, 20℃換算。
   - 出荷時3点調整の ABS は温度補正込みの `I(0)` から出るため、先に温度補正を確定。

#### パラメータ記憶
1. **プローブフラッシュの 2 領域で役割を分ける**
   - 第1領域 = PARAM: `ADC_Zero[]`, `LED_Out[]` 等。
   - 旧係数領域(廃止): 係数ページ。

2. **真実源 = プローブ flash**
   - 全係数、現場上書き分も含む。
   - 本体故障交換時も再調整不要／プローブ交換時は再調整。

3. **本体EEPROMミラー**
   - プローブ領域と完全に同一配置＋プローブ個体ID＋最終更新日（RTC由来）を保持。
   - 非権威キャッシュ。

4. **起動時 初期化（3層 + ID 判定）**
   ```
   (1) 変数に既定値をセット
   (2) 本体EEPROMミラーから読み出し上書き
   (3) 判定:
       プローブID ≠ ミラーID            → 必ずプローブから読む（別個体）
       プローブID = ミラーID かつ 日付[プローブ ≥ 本体] → プローブから読む（プローブ=真実源、同一もプローブ優先）
       プローブID = ミラーID かつ 日付[本体 > プローブ]  → 本体ミラー維持（前回プローブ書戻し失敗時のみ）→ 再同期
       プローブ読取失敗                 → 本体ミラーで測定継続（耐性）
   ```

5. **書き戻し（電源OFF 一括）**
   - 変更分を プローブ flash と 本体EEPROMミラーの両方へ、同じ ID・最終更新日で書く。
   - 校正/調整は運用中 RAM キャッシュに反映 → OFF時に一括書戻し。

6. **フラッシュ レコード割付**
   ```
   | Page | 機能 | 内容（float/32B, 特記なき限り float） |
   |---|---|---|
   | 0 | ヘッダ | magic(4B)＋ver(1)＋rsv(1)＋Probe_ID(4B)＋最終更新日(4B)＋Product_Name(7B)＋checksum(4B) |
   | 1 | プローブ共通調整 | ADZero[ch1-5]=5, LED_Out=1, k_depth=1 |
   | 2 | プローブ設定 | SADA, SADC, 選択中相関式No.(MLSS), Meas_Mode …（int/設定） |
   | 3 | MLSS ゼロ/温度 | MLSSZR_出荷時, MLSSZR_現場, refZR, B, B2 |
   | 4 | MLSS ベース/校正 | 出荷時2次(c0,c1,c2), SP_A,SP_B,SP_C, SetVal1,SetVal2 |
   | 5–9 | MLSS No.21-30 | 独立2次式×10（(c0,c1,c2)を2式/ページ）＝5ページ。初期値=P4ベースのコピー |
   | 10 | SS ゼロ/温度 | SSZR_出荷時, SSZR_現場, refZR, B, B2 |
   | 11 | SS ベース/校正 | 出荷時2次(c0,c1,c2), SP_A,B,C, SetVal1,2 |
   | 12 | SS No.21 | 独立2次式×1 (c0,c1,c2) ＋予備 |
   | 13 | TR ゼロ/ベース/校正 | TRZR, 累乗a,b, SP_A,B,C, SetVal1,2（温度補正=SS参照） |
   | 14 | TR No.21 ＋予備 | 累乗a,b(校正後) ＋予備 |
   ```

#### 調整順序
1. **LED PWM 調整**
2. **ADZero 個別オフセット**
3. **Ref 温度補正**
4. **出荷時3点調整**

#### 注意事項
- 出荷時2次式データ未取得のため、No.1-20 は基準=Mode_CF[0]（現行=線形21315）で当面 従来同等の出力。
- 出荷時3点調整(AMC/AMCF)実施後に本来の2次カーブに。
