## A. プローブ / 本体 責務分界（設計原則, 2026-07-20 確定）

- **プローブ = AD 値(mV)をそのまま出力**する。補正・計算は **本体** で行う。
- 係数・パラメータのうち **プローブに関するものはプローブ側フラッシュに保存**するが、**補正/計算は本体**。
- **プローブ側の計算に使う調整パラメータは 2 種のみ**:
  1. **`ADZero[]`** … 空中出力を 1750mV にオフセット（第1領域 PARAM）
  2. **LED PWM duty (`LED_Out[0]`)** … 初期値 0.36（第1領域 PARAM）
- **傾き span（`ADC_Span` による `×SPAN_INI/Span` 正規化）は廃止**。プローブ側計算は ADZero のみ。

改訂後のプローブ処理（実質）:
```
プローブ出力 mV = 生mV(digit×LSB) − ADZero          （LED は duty で駆動するだけ）
本体側        : 出力mV → ABS → 4次式(Mode_CF) → 温度補正 → 機差kiza → スパンSP → 物理量
```
- Mode_CF(4次) 等の**係数はプローブ vault(第2領域)に保存**するが、**適用計算は本体**（現状どおり）。

> 本原則により、以下 §1 の現行正規化式は「`×ADC_SPAN_INI/ADC_Span` を撤去し ADZero 減算のみ」に改める。

## A-2. 真実源・同期・初期化モデル（2026-07-20 確定、ID-200 の 1Wire EEPROM 流用）

- **真実源 = プローブ flash**（全係数、**現場上書き分も含む**）。本体故障交換時も再調整不要／プローブ交換時は再調整。
- **本体EEPROMミラー**: プローブ領域と**完全に同一配置**＋ **プローブ個体ID** ＋ **最終更新日**（RTC由来）を保持。非権威キャッシュ。
- **本体側は LED PWM / ADZero など計算未使用の第1領域パラメータも読み出す**（プローブ個別調整値の確認用に共通領域として保持）。
- **プローブ個体ID は既存コマンドで実装済み**（新規不要）: 書=`SID,N`（uint32 10進）／ 読=`RPP` ダンプ中の `Probe ID:` ／ 永続=`WPP`。
  `Probe_ID`(`uint32_t`) は第1領域 PARAM に格納。本体ミラーにこの ID を持たせて突合する。

**起動時 初期化（3層 + ID 判定）**:
```
(1) 変数に既定値をセット
(2) 本体EEPROMミラー（同一配置＋プローブID＋最終更新日）から読み出し上書き
(3) 判定:
    プローブID ≠ ミラーID            → 必ずプローブから読む（別個体。=1Wireのセンサー製造番号に相当）
    プローブID = ミラーID かつ 日付[プローブ ≥ 本体] → プローブから読む（プローブ=真実源、同一もプローブ優先）
    プローブID = ミラーID かつ 日付[本体 > プローブ]  → 本体ミラー維持（前回プローブ書戻し失敗時のみ）→ 再同期
    プローブ読取失敗                 → 本体ミラーで測定継続（耐性）
```
**書き戻し（電源OFF 一括）**: 変更分を **プローブ flash と 本体EEPROMミラーの両方**へ、**同じ ID・最終更新日**で書く。
- 校正/調整は運用中 RAM キャッシュに反映 → OFF時に一括書戻し（flash-busy 窓を測定中にまたがない＝VAULT不安定の緩和）。
- 急電断で当セッション変更が失われる点は 1Wire と同性質（通常はボタンOFFの制御シャットダウンで書戻し窓を確保）。


## 8. 基板調整 AD調整 — 改訂（2026-07-20）: プローブフラッシュ2領域で分担

プローブフラッシュ（`docs/probe-flash-map.md`）の 2 領域で役割を分ける。**SS / TR も共通の考え方**。

### プローブ／出荷時 調整の順序（2026-07-20 確定）
1. **LED PWM 調整**（明るさ・ch1〜4 max→1750）
2. **ADZero[] 個別オフセット**（ch1〜5 → 1750）
3. **Ref 温度補正**（5/20/35℃ 清水 Ref → 2次式, 20℃換算）← §8-T
4. **出荷時3点調整**（ABS→mg/L の **2次式** = No.1 基準）← 温度補正確定後に実施

理由（3→4 の順）: 出荷時3点調整の ABS は温度補正込みの `I(0)`（§2）から出るため、
**先に温度補正を確定**して各点 ABS を 20℃換算した状態で焼き込む。逆順は非20℃で焼いてしまいズレる。

### 第0ステップ: LED PWM 調整（＝1750mV 収束機能, ALDA 相当）
- **目的**: 空中で **ch1〜4 の AD が 2048mV（＝16bit AD 満スケール, 振り切り）に達しない**よう LED 出力を下げる。
- **基準**: **ch1〜4 の生 mV で最も高いものが 1750mV** になるよう LED duty を調整（比例＋収束反復のサーボ）。
- **保存先**: `LED_Out[0]`（第1領域 PARAM）。
- これが「1750mV に収束する機能」（先に AD0 と混同していたもの。**別機能**）。
- 実装参照: 本体 [Adj_LED_Auto](../../IM-110/Core/Src/IM_110.c#L1553)（`ALDA`）。
- **対象 ch = ch1〜4（code index `[0]`〜`[3]`）**＝ MLSS受光/MLSS_Ref/SS受光/SS_Ref。**ch5(透視度 `[4]`)は除外**（§0 対応表）。


| 領域 | ページ | 保持するもの | 本 AD 調整での役割 |
|---|---|---|---|
| **第1領域 = PARAM** | Page62 (`PARAM_AD`) | `ADC_Zero[]`, `LED_Out[]` 等 | **ADZero = 空気中出力を 1750mV にオフセットする値**（①・現状維持） |
| **第2領域 = VAULT** | Page63 (`VAULT_AD`) | 校正係数 vault | **出荷時2次式(No.1基準)**＋**出荷時清水中オフセット(MLSSZR初期値, ②)**＋Ref温度補正＋No.21-30 |

> 受光ゼロは 2 層併用（① 空中1750=ADZero / ② 清水ゼロ=MLSSZR）。詳細は §2 改訂。暗時 ADZR は廃止。

### 第1領域: ADZero（＝画面「MLSS AD0」）
- **ADZero** = **空気中**で受光出力を **1750mV にオフセット**する値。プローブフラッシュ**第1領域(PARAM)**。**今のまま**。
- **対象 ch = ch1〜5（index `[0]`〜`[4]`）を個別に**。ch5(透視度)も A/DC 別途入力ゆえ ADZero 要（LED PWM は ch1〜4 のみ）。
- ID-200T ADZERO（[Calc.c:432](../../IM-110/Core/Src/Calc.c#L432) `calc20` case0 `ADzr[0]=R`）と同操作。ID-200T は **0mV**、IM-110 は **1750mV** にオフセット。
- 取得ゲート: **生 mV 1400〜2048mV**（範囲外は取得しない＝NG）。
- 画面タイトル **「MLSS AD0」**（基板調整側。校正側「MLSS 0」とは別物）。

### 廃止
- **`ADC_Span[]`・`ADMiddle[]` は無し（廃止・未使用）**。
  → プローブ正規化の傾き補正（`× ADC_SPAN_INI / ADC_Span`, `IM_110.c:205`）は使わない方向。
  傾き/曲線は第2領域の 4次式（出荷時3点校正）が担う。

## 12. フラッシュ レコード割付（2026-07-20 確定案）

**方針**: 1 機能 = 32B ページ単位で分離。**32B 共通レコード**で プローブflash ⇄ 本体EEPROMミラー を **相対レコードN で完全一致**
（`本体EEPROM[ミラー基準+N] == プローブflash[領域先頭+N]`）。絶対ページNo.は別（本体EEPROMは header/共通設定/界面/測定履歴 等
本体固有データも持つため、ミラーは連続サブブロックに置く）。既存の本体EEPROM疎レイアウト(page19-60,2page/相関式)は本配置に**置換**。

**採用した設計判断**:
- **k_No（No.1-20 ゲイン）は全プローブ共通の固定 progression ＝ flash 非保存**（FW 定数表）。保存は「選択中 No.」のみ。
- **MLSSZR 等ゼロ基準は「出荷時」＋「現場」の 2 枠**（初期化で出荷時へ復帰）。No.21-30 は各スロットに独立2次式（初期値=出荷時ベース P4 のコピー）。

| Page | 機能 | 内容（float/32B, 特記なき限り float） |
|---|---|---|
| 0 | ヘッダ | magic(4B)＋ver(1)＋rsv(1)＋**Probe_ID**(4B)＋**最終更新日**(4B)＋Product_Name(7B)＋checksum(4B) |
| 1 | プローブ共通調整 | ADZero[ch1-5]=5, LED_Out=1, k_depth=1 |
| 2 | プローブ設定 | SADA, SADC, 選択中相関式No.(MLSS), Meas_Mode …（int/設定） |
| 3 | MLSS ゼロ/温度 | MLSSZR_出荷時, MLSSZR_現場, refZR, B, B2 |
| 4 | MLSS ベース/校正 | 出荷時2次(c0,c1,c2), SP_A,SP_B,SP_C, SetVal1,SetVal2 |
| 5–9 | MLSS No.21-30 | 独立2次式×10（(c0,c1,c2)を2式/ページ）＝5ページ。初期値=P4ベースのコピー |
| 10 | SS ゼロ/温度 | SSZR_出荷時, SSZR_現場, refZR, B, B2 |
| 11 | SS ベース/校正 | 出荷時2次(c0,c1,c2), SP_A,B,C, SetVal1,2 |
