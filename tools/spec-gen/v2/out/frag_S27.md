「年設定」画面は、IM-110Tの年設定を行うためのインターフェースを提供します。以下に、この画面の動作仕様を詳細に記述します。

### 表示要素
1. **タイトル**:
   - 画面上部中央に「年設定」と表示されます。
2. **アイコン表示**:
   - 画面左側に「DISP」アイコンと「MEM」アイコンが表示されます。
3. **編集状態のアイコン**:
   - `en`引数が有効 (`en == 1`) の場合、選択中のアイコンは「選択」と「決定」になります。
   - `en`引数が無効 (`en == 0`) の場合、選択中のアイコンは「次へ」と「開始」になります。
4. **操作アイコン**:
   - 「+」アイコン: `select`が0で`fl`が1のときに点灯します。
   - 「-」アイコン: `select`が1で`fl`が1のときに点灯します。
   - 「中止」アイコン: `select`が2で`fl`が1のときに点灯します。
   - 「セット」アイコン: `select`が3で`fl`が1のときに点灯します。
5. **年表示**:
   - 画面中央に現在設定されている年が数字で表示されます。

### 引数の効果
- **batvol**: 電池残量を表し、画面上部に表示されます。
- **fl**: アイコンの点滅制御を行います。`select`と組み合わせて特定のアイコンを点灯させるかどうかを決定します。
- **en**: 編集状態の有無を示します。編集可能な場合 (`en == 1`) は「選択」と「決定」アイコンが表示され、編集不可な場合 (`en == 0`) は「次へ」と「開始」アイコンが表示されます。
- **select**: 現在選択されている操作を示します。`select`の値に応じて、「+」、「-」、「中止」、「セット」のいずれかのアイコンが点灯します。

### 状態ごとの差異
- **編集可能 (`en == 1`) の場合**:
  - 「選択」と「決定」アイコンが表示されます。
  - `select`に応じて、「+」、「-」、「中止」、「セット」のいずれかのアイコンが点灯します。
- **編集不可 (`en == 0`) の場合**:
  - 「次へ」と「開始」アイコンが表示されます。
  - `select`に応じて、「+」、「-」、「中止」、「セット」のいずれかのアイコンが点灯します。

### 注意事項
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）で定義されているため、本コードには含まれていません。これらの詳細は要確認です。

この仕様に基づき、「年設定」画面はユーザーが年を設定するための直感的なインターフェースを提供します。
