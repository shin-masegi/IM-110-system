### 月日設定画面の動作仕様

#### 共通表示
- タイトル: 最上段に「月日設定」が表示される。
- 日付時刻: 最上段に現在の日付と時刻が表示される。
- 電池残量: 最上段に電池残量が表示される。`batvol` 引数によって表示内容が変わる。
- BT接続アイコン: Bluetooth接続中は最上段にBTアイコンが表示される。

#### 編集状態の表示 (`en` が 1 の場合)
- 「選択」アイコン: (5, 18) に表示。
- 「決定」アイコン: (5, 23) に表示。

#### 編集前の表示 (`en` が 0 の場合)
- 「次へ」アイコン: (5, 18) に表示。
- 「開始」アイコン: (5, 23) に表示。

#### 月日設定
- 月の表示:
  - 十の位: (16, 6) に表示。`month / 10 % 10` の値が表示される。
  - 一の位: (22, 6) に表示。`month % 10` の値が表示される。
- 日付の区切り: (28, 6) に「／」が表示される。
- 日の表示:
  - 十の位: (32, 6) に表示。`day / 10 % 10` の値が表示される。
  - 一の位: (38, 6) に表示。`day % 10` の値が表示される。

#### アイコンの点滅制御 (`fl` が 1 の場合)
- 「+」アイコン:
  - 月の十の位: (16, 2) に `select == 0` のとき黒、それ以外は白。
  - 月の一の位: (16, 18) に `select == 1` のとき黒、それ以外は白。
- 「-」アイコン:
  - 月の十の位: (22, 2) に `select == 2` のとき黒、それ以外は白。
  - 月の一の位: (22, 18) に `select == 3` のとき黒、それ以外は白。
- 「+」アイコン:
  - 日の十の位: (32, 2) に `select == 4` のとき黒、それ以外は白。
  - 日の一の位: (32, 18) に `select == 5` のとき黒、それ以外は白。
- 「-」アイコン:
  - 日の十の位: (38, 2) に `select == 6` のとき黒、それ以外は白。
  - 日の一の位: (38, 18) に `select == 7` のとき黒、それ以外は白。

#### その他アイコン
- 「中止」アイコン: (28, 22) に表示。`select == 8` のとき黒、それ以外は白。
- 「セット」アイコン: (39, 22) に表示。`select == 9` のとき黒、それ以外は白。

#### 引数の効果
- `batvol`: 電池残量を示す値。最上段に表示される。
- `fl`: アイコンの点滅制御。`fl == 1` の場合、選択されているアイコンが黒で表示され、それ以外は白で表示される。
- `select`: 現在選択されている項目を示す値。`select` に応じて対応するアイコンが点滅する。
- `en`: 編集状態の有効/無効を示す値。`en == 1` の場合は編集状態、`en == 0` の場合は編集前の表示となる。

#### 注意事項
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW (Normal.c/Setting.c/Adjust.c) で定義されているため、本コードには含まれていない。
