測定メニューは、MLSSメーターIM-110Tの主要な操作画面であり、以下の要素を表示します。

### 共通表示
- タイトル: 最上段に「測定メニュー」が表示されます。
- 日付時刻: 最上段に現在の日付と時刻が表示されます。
- 電池残量アイコン: 最上段に電池残量を示すアイコンが表示されます。`batvol`引数によって表示内容が変わります。
- BT接続中はBTアイコンも表示されます。

### メニュー項目
- 「測定モード」: `select == 0`かつ`fl == 1`の場合、点滅します。界面測定モード（Meas_Mode == MEAS_MODE_DEPTH）の場合、「界面設定」アイコンに差し替えられます。
- 「相関式」: `select == 1`かつ`fl == 1`の場合、点滅します。界面測定モードでは「界面設定」アイコンに差し替えられます。
- 「校正」: `select == 2`かつ`fl == 1`の場合、点滅します。
- 「設定」: `select == 3`かつ`fl == 1`の場合、点滅します。

### アイコン
- 「戻る」アイコン: 長押しで前の画面に戻ります（要確認: 呼出側FW）。
- 「選択」アイコン: メニュー項目を選択するためのアイコンです。
- 「決定」アイコン: 選択したメニュー項目を決定するためのアイコンです。

### 引数の効果
- `select`: メニュー項目の選択状態を示します。`0`から`3`までの値を取り、対応するメニュー項目が点滅します。
- `batvol`: 電池残量を示すアイコンの表示内容を決定します。
- `fl`: メニュー項目の点滅状態を制御します。`1`の場合、選択されたメニュー項目が点滅します。

### 画面遷移
- 長押しで前の画面に戻る（要確認: 呼出側FW）。
- 各アイコンを選択して決定することで、対応する機能への遷移が行われます（要確認: 呼出側FW）。

この画面は、MLSSメーターIM-110Tの主要な操作を行うためのインターフェースであり、ユーザーが測定モードや設定を選択・決定するためのものです。
