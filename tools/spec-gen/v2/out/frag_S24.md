IM-110Tの「淡水/海水設定」画面は、モノクロLCD 400×240に表示され、以下の要素と動作を持つ。

### 共通表示
- **タイトル**: 最上段に「淡水/海水設定」が表示される。
- **日付時刻**: タイトルの右側に現在の日付と時刻が表示される。
- **電池残量アイコン**: 最上段の右端に電池残量を示すアイコンが表示される。`batvol`引数によって表示内容が変わる。
- **BT接続中アイコン**: Bluetooth接続中は、最上段にBTアイコンが表示される。

### 操作アイコン
- **選択アイコン**: 画面の左側中央付近に「選択」を示すアイコンが表示される。
- **決定アイコン**: 画面の右側中央付近に「決定」を示すアイコンが表示される。

### 設定項目
- **淡水/海水選択**:
  - `flg`引数によって表示内容が変わる。
    - `flg == 0`: 淡水と海水の選択肢が表示される。`select`引数と`fl`引数によって、選択中の項目が強調表示される。
      - `select == 0 && fl == 1`: 「淡水」が強調表示される。
      - `select == 1 && fl == 1`: 「海水」が強調表示される。
    - `flg == 1`: 「淡水設定中」というメッセージが表示される。
    - `flg == 2`: 「海水設定中」というメッセージが表示される。

### 引数の効果
- **batvol**: 電池残量を示すアイコンの表示内容を決定する。具体的な電池残量に応じたアイコンが表示される。
- **flg**: 画面の表示モードを制御する。
  - `0`: 淡水と海水の選択肢を表示。
  - `1`: 「淡水設定中」メッセージを表示。
  - `2`: 「海水設定中」メッセージを表示。
- **select**: 現在選択されている項目を示す。`0`が淡水、`1`が海水を示す。
- **fl**: 点滅表示制御。`1`の場合、選択中の項目が強調表示される。

### 注意事項
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）に依存するため、本コードには記載されていない。これらの情報は『(要確認: 呼出側FW)』と明記する必要がある。
