「水温校正」画面は、MLSSメーターIM-110Tの水温校正設定を行うためのインターフェースを提供します。以下に、この画面の動作仕様を詳細に説明します。

### 表示要素
- **タイトル**: 最上段に「水温校正」と表示されます。
- **アイコン**:
  - `icon_DISP` (座標: 0, 18)
  - `icon_MEM` (座標: 0, 23)
  - 編集状態:
    - `icon_select` (座標: 5, 18)
    - `icon_decision` (座標: 5, 23)
  - 編集前状態:
    - `icon_next` (座標: 5, 18)
    - `icon_start` (座標: 5, 23)
- **操作アイコン**:
  - `+` (座標: 44, 10): `select == 0`かつ`fl == 1`の場合は黒色、それ以外は白色。
  - `-` (座標: 44, 16): `select == 1`かつ`fl == 1`の場合は黒色、それ以外は白色。
  - 初期化 (座標: 17, 21): `select == 2`かつ`fl == 1`の場合は黒色、それ以外は白色。
  - 中止 (座標: 28, 21): `select == 3`かつ`fl == 1`の場合は黒色、それ以外は白色。
  - セット (座標: 39, 21): `select == 4`かつ`fl == 1`の場合は黒色、それ以外は白色。
- **水温表示**: 水温値が数字で表示されます（座標: 37, 9）。

### 引数の効果
- **batvol**: 電池残量を表示します。具体的な表示位置や形式はコードから読み取れません。
- **fl**: アイコンの点滅制御を行います。`select`が特定の値の場合に、対応するアイコンが黒色（選択中）または白色（未選択）で表示されます。
- **en**:
  - `en == 1`の場合: 編集状態として、`icon_select`と`icon_decision`が表示されます。
  - `en == 0`の場合: 編集前状態として、`icon_next`と`icon_start`が表示されます。
- **select**: 選択中の操作を示します。以下のようにアイコンが変化します。
  - `select == 0`: `+`アイコンが黒色で点滅。
  - `select == 1`: `-`アイコンが黒色で点滅。
  - `select == 2`: 初期化アイコンが黒色で点滅。
  - `select == 3`: 中止アイコンが黒色で点滅。
  - `select == 4`: セットアイコンが黒色で点滅。

### 状態ごとの差異
- **編集状態 (`en == 1`)**:
  - `icon_select`と`icon_decision`が表示されます。
  - `+`、`-`、`初期化`、`中止`、`セット`アイコンが選択に応じて黒色で点滅します。

- **編集前状態 (`en == 0`)**:
  - `icon_next`と`icon_start`が表示されます。
  - `+`、`-`、`初期化`、`中止`、`セット`アイコンが選択に応じて黒色で点滅します。

### その他
- **画面遷移条件やスイッチ割当の詳細**については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）に依存するため、本コードからは読み取れません。これらの情報は別途確認が必要です。

以上が「水温校正」画面の動作仕様です。
