# MLSS測定画面（待機・測定中）

## 共通表示
- **タイトル**: 最上段に「MLSS」と表示。
- **日付時刻**: 最上段に現在の日付と時刻を表示。
- **電池残量**: 最上段に電池残量アイコンを表示。
- **BT接続中**: BT接続中は最上段にBTアイコンを表示。

## MLSS測定画面（待機）
### 表示要素
- **MLSS値**:
  - 正常範囲内の場合: MLSS値を10単位で四捨五入して表示。10000 mg/L以上はg/L形式で表示。
  - 異常値の場合: 白塗りで点滅表示。
- **単位**:
  - 10000 mg/L以上: "g/L"と表示。
  - それ以外: "mg/L"と表示。
- **界面判断バー**: MLSS_inst / Interface_Thresholdを100%としてバー表示。右端が118.75%に達するまで伸縮。
- **水深表示**:
  - 現在のDepth (Sサイズ): bar3の右隣にリアルタイムで表示。
  - Interface_Hold (Mサイズ): 右上に表示。未捕捉時は"-.--"と表示。
- **モードラベル**: 「モード/30」と表示。

### 引数の効果
- `flash_flg`:
  - 1: MLSS値が異常範囲内の場合、白塗りで点滅表示。
  - 0: 通常表示。
- `batvol`: 電池残量アイコンの表示に使用。

## その他
- **進捗バー**: 測定進行状況を示すバーを表示。
- **切替アイコン**: 画面右上に「DISP」と「切替」アイコンを表示。
- **保存アイコン**:
  - `save == 0`: 「MEM」と「SAVE」アイコンを表示。
  - `save == 1`: 「SAVE」と「COMPLETE」アイコンを表示。
  - `save == 2`: 「RELEASE」と「COMPLETE」アイコンを表示。
