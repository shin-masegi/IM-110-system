校正設定画面では、以下の要素が表示されます。

最上段にはタイトル「t_cal_mode」が表示されます。また、電池残量アイコンが表示され、これは `batvol` 引数に基づいて更新されます。日付時刻も常に表示されます。

画面の中央部分には、「DISP」と「MEM」のアイコンが表示されます。これらは固定で表示されます。

選択可能な項目として、「ゼロ校正」「2点校正」「3点校正」があり、それぞれのアイコンが表示されます。`select` 引数に基づいて、現在選択されている項目のアイコンが強調表示（点滅）されます。具体的には、`select == 0` の場合は「ゼロ校正」、`select == 1` の場合は「2点校正」、`select == 2` の場合は「3点校正」のアイコンが点滅します。点滅の有無は `fl` 引数で制御され、`fl == 1` のときに点滅表示となります。

選択と決定のためのアイコンも表示され、これらは固定で表示されます。

LCDの更新が正常に行われない場合、関数は `DISP_NG` を返し、終了します。
