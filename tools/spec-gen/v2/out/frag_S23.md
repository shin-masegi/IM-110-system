画面「自動校正設定」は、IM-110TのモノクロLCDに表示される設定画面であり、以下の要素を含む。

### 共通表示
- **タイトル**: 最上段に「自動校正設定」と表示。
- **日付時刻**: タイトルの右側に現在の日付と時刻が表示。
- **電池残量アイコン**: 電池残量に応じたアイコンが最上段に表示。`batvol`引数によって表示内容が変わる。
- **BT接続中アイコン**: Bluetooth接続中は、BTアイコンが表示される。

### 固定表示
- **DISPアイコン**: 座標(0, 18)に「DISP」アイコンを表示。
- **MEMアイコン**: 座標(0, 23)に「MEM」アイコンを表示。
- **選択アイコン**: 座標(5, 18)に「選択」アイコンを表示。
- **決定アイコン**: 座標(5, 23)に「決定」アイコンを表示。

### 動的表示
- **自動校正設定オプション**:
  - `flg`引数によって表示内容が変わる。
    - `flg == 0`: 選択可能なオプションが表示される。
      - 「OFF」: 座標(18, 7)に表示。`select == 0`かつ`fl == 1`の場合は強調表示。
      - 「4時」: 座標(34, 7)に表示。`select == 1`かつ`fl == 1`の場合は強調表示。
      - 「5時」: 座標(18, 18)に表示。`select == 2`かつ`fl == 1`の場合は強調表示。
      - 「6時」: 座標(34, 18)に表示。`select == 3`かつ`fl == 1`の場合は強調表示。
    - `flg == 1`: 「OFF設定中」メッセージが座標(15, 6)に表示。
    - `flg == 2`: 「4時設定中」メッセージが座標(15, 6)に表示。
    - `flg == 3`: 「5時設定中」メッセージが座標(15, 6)に表示。
    - `flg == 4`: 「6時設定中」メッセージが座標(15, 6)に表示。

### 引数の効果
- **`select`**: 選択されているオプションを示す。`0`から`3`までの値を取り、対応するオプションが強調表示される。
- **`flg`**: 設定中の状態を示す。`0`以外の値では特定のメッセージが表示される。
- **`batvol`**: 電池残量に応じたアイコンが表示される。
- **`fl`**: オプションの強調表示制御。`1`の場合、選択されているオプションが強調表示される。

### 注意事項
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）に依存するため、本コードからは読み取れない。これらの情報は『(要確認: 呼出側FW)』と明記する。
