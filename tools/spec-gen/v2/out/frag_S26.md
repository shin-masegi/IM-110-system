時刻設定画面は、IM-110Tの時刻設定を行うためのインターフェースを提供します。以下に、この画面の動作仕様を詳細に記述します。

### 表示要素
- **タイトル**: 最上段に「時刻設定」と表示されます。
- **電池残量アイコン**: 現在の電池残量を示すアイコンが表示されます。
- **操作アイコン**:
  - 「DISP」アイコン: (0, 18)座標に表示。
  - 「MEM」アイコン: (0, 23)座標に表示。
  - 「選択」アイコン: (5, 18)座標に表示。
  - 「決定」アイコン: (5, 23)座標に表示。
- **操作ボタン**:
  - 「+」ボタン: (44, 10)座標に表示。選択状態で点滅します。
  - 「-」ボタン: (44, 16)座標に表示。選択状態で点滅します。
  - 「セット」ボタン: (28, 21)座標に表示。選択状態で点滅します。
  - 「中止」ボタン: (39, 21)座標に表示。選択状態で点滅します。
- **時刻表示**:
  - 年月日: (17, 5)座標に表示。
  - 時: (15, 9)と(21, 9)座標に表示。コロン「:」が挿入されます。
  - 分: (31, 9)と(37, 9)座標に表示。

### 引数の効果
- **batvol**: 電池残量を示す値で、電池残量アイコンの表示内容を決定します。
- **fl**: 点滅表示制御フラグ。選択されている操作ボタンが点滅するかどうかを制御します。
- **select**: 現在選択されている操作ボタンを示す値。以下のように動作します。
  - `select == 0`: 「+」ボタンが選択され、点滅表示されます。
  - `select == 1`: 「-」ボタンが選択され、点滅表示されます。
  - `select == 2`: 「セット」ボタンが選択され、点滅表示されます。
  - `select == 3`: 「中止」ボタンが選択され、点滅表示されます。

### 状態ごとの差異
- **選択状態**:
  - 選択されている操作ボタンは点滅表示されます。他のボタンは通常表示です。
- **電池残量**:
  - 電池残量アイコンは、batvolの値に応じて異なるアイコンが表示されます。

### 注意事項
- 画面内の「開始」等はアイコン表示であってタッチボタンではないため、操作は物理スイッチを使用します。
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）を参照してください。
