「測定履歴表示」画面は、MLSSメーターIM-110TのモノクロLCD（400×240）に表示される画面であり、以下の要素と動作仕様を持つ。

### 表示要素
1. **タイトル**: 最上段に「測定履歴」というタイトルが表示される。
2. **日付時刻**: タイトルの右側に現在の日付と時刻が表示される。
3. **電池残量アイコン**: 最上段の右端に電池残量を示すアイコンが表示される。`batvol`引数によってアイコンの表示内容が変わる。
4. **スイッチ表示**:
   - `index < 3`の場合:
     - `erase`が1のとき、左上に「消去」アイコンが表示される。
     - `erase`が0のとき、左上に「表示」アイコンが表示される。
     - 左下に「メモリ」アイコンが表示される。
     - 右上に「測定」アイコンが表示される。
     - 右下に「次へ」アイコンが表示される。
   - `index >= 3`の場合:
     - 左上に「表示」アイコンが表示される。
     - 左下に「メモリ」アイコンが表示される。
     - 右上に「測定」アイコンが表示される。
     - 右下に「次へ」アイコンが表示される。
5. **履歴データ**:
   - `index`番目の履歴データが16行目、6列目から表示される。
   - `index + 1`番目の履歴データが16行目、14列目から表示される。
   - `index + 2`番目の履歴データが16行目、22列目から表示される。
   - 各履歴データの間に区切り線（アイコン）が表示される。

### 引数の効果
- **batvol**: 電池残量を示す値。この値によって電池残量アイコンの表示内容が変わる。
- **erase**: 消去操作の有無を示すフラグ。`index < 3`の場合にのみ効果があり、1のときは「消去」アイコンが、0のときは「表示」アイコンが表示される。
- **index**: 履歴データのインデックス。この値によって表示される履歴データが変わる。

### 状態ごとの差異
- `index < 3`の場合:
  - 左上に「消去」または「表示」アイコンが表示される。
  - 右上に「測定」アイコンが表示される。
  - 右下に「次へ」アイコンが表示される。
- `index >= 3`の場合:
  - 左上に常に「表示」アイコンが表示される。
  - 右上に「測定」アイコンが表示される。
  - 右下に「次へ」アイコンが表示される。

### 注意事項
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）で定義されているため、本コードには含まれていない。これらの詳細は要確認: 呼出側FW。
