「基板調整：プログラムVer表示」画面は、IM-110Tのプログラムバージョンを表示するための画面である。以下にその動作仕様を詳細に記述する。

### 表示要素
1. **タイトル**:
   - `LS027_disp_icon(0, 0, (uint8_t *)t_prog_Ver);` により、画面の左上（座標(0, 0)）に「プログラムVer」というタイトルが表示される。

2. **プログラムバージョン**:
   - `LS027_disp_number(36, 9, 0, (float)FW_VER1 / 100.0, 2);` により、座標(36, 9)にプログラムのバージョンが表示される。バージョンは`FW_VER1`から取得され、小数点以下2桁まで表示される。

3. **DISPアイコン**:
   - `LS027_disp_icon(0, 17, (uint8_t *)icon_DISP);` により、座標(0, 17)に「DISP」というアイコンが表示される。

4. **次へアイコン**:
   - `LS027_disp_icon(5, 17, (uint8_t *)icon_next);` により、座標(5, 17)に「次へ」というアイコンが表示される。

5. **メモリ消去関連アイコン**:
   - `del_flag`の値によって異なるアイコンが表示される。
     - `del_flag`が0の場合：
       - `LS027_disp_icon(0, 22, (uint8_t *)icon_MEM);` により、座標(0, 22)に「MEM」というアイコンが表示される。
       - `LS027_disp_icon(5, 22, (uint8_t *)icon_l_erase);` により、座標(5, 22)に「長押しで消去」というアイコンが表示される。
     - `del_flag`が1の場合：
       - `LS027_disp_icon(0, 22, (uint8_t *)icon_erase);` により、座標(0, 22)に「消去完了」というアイコンが表示される。

### 引数の効果
- **del_flag**:
  - `del_flag`が0の場合、メモリ消去関連のアイコンが「MEM」と「長押しで消去」として表示される。
  - `del_flag`が1の場合、「消去完了」というアイコンが表示される。

### その他
- **LCD初期化**:
  - `lcd_init(0)`により、LCDの初期化と表示バッファの更新状態を確認する。初期化に失敗した場合は`DISP_NG`を返し、処理を終了する。

- **LCD表示更新**:
  - `SPI_transmitLCD();`により、LCDへの表示内容が実際に送信され、画面に反映される。

### 注意点
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）で定義されているため、本コードからは読み取れない。これらの情報は『(要確認: 呼出側FW)』と明記する必要がある。

以上が「基板調整：プログラムVer表示」画面の動作仕様である。
