「測定モード設定」画面は、MLSSメーターIM-110Tの測定モードを設定するための画面である。以下に、この画面の動作仕様を詳細に記述する。

### 表示要素
1. **タイトル**: 最上段に「測定モード設定」と表示される。
2. **日付時刻**: タイトルの右側に現在の日付と時刻が表示される。
3. **電池残量アイコン**: タイトルの左側に電池残量を示すアイコンが表示される。`batvol`の値によってアイコンの表示内容が変わる。
4. **モード選択アイコン**:
   - 「通常」モード: `select == 0`かつ`fl == 1`の場合は青色で表示、`それ以外`の場合は白色で表示される。
   - 「ロガー」モード: `select == 1`かつ`fl == 1`の場合は青色で表示、`それ以外`の場合は白色で表示される。
5. **決定アイコン**: 最下段に「決定」を示すアイコンが表示される。

### 引数の効果
- `select`: モード選択の位置を示す。0の場合は「通常」モード、1の場合は「ロガー」モードが選択されていることを示す。
- `flg`: 画面の表示内容を制御するフラグ。
  - `0`またはデフォルト: モード選択画面を表示。`select`と`fl`の値によって「通常」と「ロガー」のアイコンが青色か白色かになる。
  - `1`: 「通常モード設定中」というメッセージを表示。
  - `2`: 「ロガーモード設定中」というメッセージを表示。
- `batvol`: 電池残量を示す値。この値に基づいて電池アイコンが表示される。
- `fl`: 点滅制御フラグ。`select`と組み合わせて、選択中のモードアイコンを点滅させる。

### 状態ごとの差異
- **通常モード選択時**:
  - `select == 0`かつ`fl == 1`: 「通常」モードが青色で表示される。
  - `select != 0`または`fl != 1`: 「通常」モードが白色で表示される。
- **ロガーモード選択時**:
  - `select == 1`かつ`fl == 1`: 「ロガー」モードが青色で表示される。
  - `select != 1`または`fl != 1`: 「ロガー」モードが白色で表示される。
- **設定中メッセージ**:
  - `flg == 1`: 「通常モード設定中」というメッセージが表示される。
  - `flg == 2`: 「ロガーモード設定中」というメッセージが表示される。
