モード切替画面は、MLSSメーターIM-110Tの測定モードを選択するためのインターフェースです。以下に、この画面の動作仕様を詳細に説明します。

### 表示要素
1. **タイトル**:
   - 最上段に「t_measure」というタイトルが表示されます。
2. **電池残量アイコン**:
   - `batvol` 引数に基づいて、電池の残量を示すアイコンが表示されます。
3. **日付時刻**:
   - 最上段に現在の日付と時刻が表示されます。
4. **モード選択アイコン**:
   - `LS027_disp_icon` 関数を使用して、以下のアイコンが表示されます。
     - (0, 18) に `icon_DISP`
     - (0, 23) に `icon_MEM`
5. **選択・決定アイコン**:
   - `LS027_disp_icon` 関数を使用して、以下のアイコンが表示されます。
     - (5, 18) に `icon_select`（選択）
     - (5, 23) に `icon_decision`（決定）

### モード切替
モード切替は、`flg` 引数に基づいて以下のように表示されます。

- **flg == 0**:
  - `select` と `fl` 引数に基づいて、以下のアイコンが表示されます。
    - (18, 7) に `b_MLSS` または `w_MLSS`（選択時は `b_MLSS`、非選択時は `w_MLSS`）
    - (34, 7) に `b_SS` または `w_SS`（選択時は `b_SS`、非選択時は `w_SS`）
    - (18, 18) に `b_Transparency` または `w_Transparency`（選択時は `b_Transparency`、非選択時は `w_Transparency`）

- **flg == 1**:
  - (15, 6) に `msg_MLSS` が表示されます。

- **flg == 2**:
  - (15, 6) に `msg_SS` が表示されます。

- **flg == 3**:
  - (15, 6) に `msg_Transparency` が表示されます。

- **flg == 4**:
  - (15, 6) に `msg_Interface` が表示されます。

### 引数の効果
- **batvol**: 電池残量アイコンの表示に使用されます。
- **select**: モード選択時のアイコンの色を変更します。選択されているモードは強調表示（`b_MLSS`, `b_SS`, `b_Transparency`）され、非選択のものは通常表示（`w_MLSS`, `w_SS`, `w_Transparency`）されます。
- **fl**: 点滅表示制御に使用されます。`fl == 1` の場合、選択されているモードが点滅します。

### 注意事項
- 「界面」アイコンは廃止されており、設定メニュー側で界面しきい値だけ設定する仕様に移行しています。
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）で確認が必要です。

以上が、モード切替画面の動作仕様です。
