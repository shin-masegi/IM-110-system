画面「後校正 記録選択」は、モノクロLCD 400×240に表示され、以下の要素を含む。

### 共通表示
- タイトル: 「履歴」と表示される。
- 日付時刻: 現在の日付と時刻が表示される。
- 電池残量: `batvol` に基づいて電池アイコンが表示される。電池残量に応じて異なるアイコンが表示される。
- BT接続中はBTアイコンが表示される。

### スイッチ操作
- DISPスイッチ短押し: 記録番号を次へ送る（No.送り）。
- MEMスイッチ短押し: 校正を行う。
- DISPスイッチ長押し: 校正画面に戻る。

### 表示要素
- スイッチアイコン:
  - `icon_DISP` アイコンが (0, 18) に表示される。
  - `icon_next` アイコンが (5, 18) に表示され、次への操作を示す。
  - `icon_MEM` アイコンが (0, 23) に表示される。
  - `icon_measure` アイコンが (5, 23) に表示され、校正を行うことを示す。

### 記録選択
- 記録番号は最大3行分表示され、各行のY座標はそれぞれ6, 14, 22である。
- `index` と `cursor` に基づいて、現在選択されている行が点滅する。`flash_flg` が有効な場合、選択された記録番号が点滅表示される。
- 各行の区切り線は `icon_line` アイコンで表示され、各行のY座標 + 7 の位置に配置される。

### 引数の効果
- `index`: 現在表示されている最初の記録番号のインデックスを示す。
- `cursor`: 現在選択されている記録番号のインデックスを示す。
- `count`: 総記録数を示す。`index + r` が `count` を超えない範囲で記録が表示される。
- `flash_flg`: 選択された行の点滅制御を行う。有効な場合、選択された記録番号が点滅する。
- `batvol`: 電池残量に基づいて電池アイコンを表示する。

### 状態ごとの差異
- 選択されている行は `flash_flg` に従って点滅表示される。他の行は常に表示される。
- 記録番号が3行分表示され、各行には区切り線が表示される。

### 注意事項
- 他画面への遷移条件やスイッチ割当の詳細については、呼び出し側FW（Normal.c/Setting.c/Adjust.c）で定義されているため、本コードでは明記されていない。
