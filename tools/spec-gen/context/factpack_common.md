# IM-110 事実パック（共通・本体）

> これは仕様書生成の「与える事実」。ここに書かれた事実だけを使い、憶測でハードやコマンドを補わないこと。

## 製品アイデンティティ
- 製品名: **MLSSメーター／IM-110T**（起動画面表記）。メーカー: 飯島電子工業株式会社。
- 用途: 活性汚泥処理の水質計測。測定項目は **MLSS / SS / 透視度 / 界面**。
- 構成: **本体基板(IM-110T)** と **プローブ基板** の2基板。RS-232Cケーブルで接続し、本体がマスターとしてプローブを制御する。
- プローブはバッテリー非搭載で、本体CN4から+5V給電される。プローブ単独では動作しない。

## 本体ハードウェア（真実源: CLAUDE.md §4、回路図/NET）
- MCU: **STM32L452RET6**（Cortex-M4, 80MHz, Flash 512kB / SRAM 160kB）。
- 表示: **Sharp メモリ液晶 LS027B7**、**400×240ドット・モノクロ**。SPI1接続、CN3(10pin FPC)経由。
- 操作スイッチ: **電源(SW3)**, **記録スイッチ(SW1=PC10)**, **履歴/表示スイッチ(SW2=PC11)** の3個。パネルシート＋画面はアイコン方式。
- 電源: V_BAT → MAX8881(U9) → 3V3。5V系は LTC3221(U13) チャージポンプ → CN4経由でプローブへ給電。
- ストレージ/時計: SPI2共有バスに **R2045S RTC(U8)** と **M95256 EEPROM(U11)**。ログ・設定を保持。
- BLE: STM32 USART1 → 74_2G125バッファ(U7) → **RN4871(U5)**。Reader(スマホアプリ)とBLE通信。
- 本体⇔プローブ通信: STM32 **LPUART1**(U4.8=RX/U4.9=TX) → MAX3232(U16) → CN4 → ケーブル。

## LCD共通レイアウト（ID-200T踏襲、IM-110向けに翻案）
- 最上段: タイトル、日付時刻、電池残量アイコン。ほぼ全画面共通。BT接続中はBTアイコン表示。
- 画面更新間隔は0.5秒が基本。
- 測定モードでは点滅表示は行わない（ID-200T踏襲）。

## 測定項目とレンジ（xlsx 操作表示一覧「1．測定」より）
- 測定メニュー: (1)MLSS / (2)SS / (3)透視度 / (4)界面 の4系統。
- 校正メニュー: (1)MLSS / (2)SS / (3)透視度 / (4)校正リセット。
- 設定メニュー: (1)時刻設定 / (2)初期化 / (3)製品情報。
- ※各測定値の具体レンジ・小数桁は本体FWと突き合わせて後日校正で確定（生成時は「機種のレンジに従う」と記す）。

## スイッチ操作の一般則（ID-200T踏襲、IM-110で要確認）
- 待機画面で記録スイッチ → 測定開始 / 測定中画面へ。
- 待機画面で履歴スイッチ → 履歴表示画面へ。
- 履歴スイッチ長押し(2s) → モード/メニュー切替メッセージ2秒表示。
- ※IM-110固有の割り当ては本体FW(Core/Src/*Display*.c)と突き合わせて校正時に確定する。
